# 商品主图修改系统 - 使用说明

## 🎉 重要更新 (2025-10-24)

### ✅ 已修复：保留卖点文字和图片尺寸
### ✅ 新增：多语言支持（俄语、中文、英语等）
### ✅ 新增：复杂布局支持（信息面板）

系统现在会：
1. **✅ 保留所有产品卖点文字**（任何语言：中文、俄语、英语等）
2. **✅ 保留产品信息文字**（成分、认证、使用说明、规格等）
3. **✅ 保留信息面板**（红色或彩色侧边栏，包含所有文字）
4. **✅ 仅删除品牌Logo**（品牌名称和商标，如Nike、Янцзы等）
5. **✅ 保持原始图片尺寸**（宽度和高度完全一致，不裁剪）
6. **✅ 支持复杂布局**（左右分栏、信息面板等）

## 快速开始

### 方式一：批量上传（推荐）

1. 点击上传区域选择图片（最多50张）
2. 预览已选图片（可删除不需要的）
3. 调整修改程度（10-100%）
4. 可选：输入新Logo文字
5. 点击"批量处理"按钮

### 方式二：单张链接

1. 右键点击商品图 → "在新标签页中打开图片"
2. 复制地址栏URL（必须以.jpg/.png/.webp结尾）
3. 粘贴到输入框
4. 调整修改程度
5. 点击"开始AI修改"

## 文字处理说明

### ✅ 保留的内容（支持所有语言）
- **中文**：产品卖点（"防水"、"透气"、"轻便"、"5小时续航"）
- **俄语**：产品特性（"Нержавеющая сталь"、"1500W большая мощность"、"18L емкость"）
- **英语**：产品功能（"Waterproof"、"Lightweight"、"Fast Charging"）
- 产品规格（尺寸、重量、材质、认证）
- 信息面板（红色或彩色侧边栏的所有文字）
- 使用说明和注意事项

### ❌ 删除的内容（仅品牌）
- **中文品牌**：Nike、耐克、Adidas、阿迪达斯
- **俄语品牌**：Янцзы（品牌名）
- **英语品牌**：Apple、Samsung
- 公司Logo和商标符号

### 示例对比

**中文示例**：
- 原图：`Nike 防水透气运动鞋 轻便舒适`
- 修改后（无新Logo）：`防水透气运动鞋 轻便舒适`
- 修改后（新Logo）：`MyBrand 防水透气运动鞋 轻便舒适`

**俄语示例**：
- 原图：`Янцзы Нержавеющая сталь 1500W большая мощность`
- 修改后（无新Logo）：`Нержавеющая сталь 1500W большая мощность`
- 修改后（新Logo）：`MyBrand Нержавеющая сталь 1500W большая мощность`

**信息面板示例**：
- 左侧红色信息栏包含所有卖点文字 → **完整保留**
- 只删除栏内的品牌名称
- 保持红色背景和文字布局

## 修改程度说明

- **10-25%**: 轻微调整（光照、背景微调）
- **26-50%**: 中度修改（背景更换、光照调整）
- **51-75%**: 明显变化（全新背景、光照风格）
- **76-100%**: 大幅改变（创意背景、艺术处理）

## AI 处理流程

1. 🔍 **GPT-4o** 分析原图（区分品牌Logo和卖点文字）
2. 🎨 **Gemini 2.5** 生成修改后的图片（保留卖点，删除品牌）
3. 📊 质量检查（SSIM、pHash、边缘质量、几何一致性）
4. 🔄 如不合格自动重试（最多2次）
5. ✅ 返回最佳结果

## 质量保证

### 自动质检
- **P0致命**：产品变形/颜色改变 → 自动重试
- **P1严重**：差异不足30%/背景不符 → 自动重试
- **P2轻微**：细小瑕疵 → 可接受

### 智能重试
- 失败自动重试最多2次
- 自动调整修改强度和提示词模板
- 返回最佳质量的候选图片

## 常见问题

### Q: 卖点文字会被删除吗？
**A**: 不会！只删除品牌Logo（如Nike、Adidas、Янцзы），所有功能描述都完整保留，支持任何语言。

### Q: 支持俄语商品图吗？
**A**: 完全支持！系统支持：
- 中文、俄语、英语等所有语言
- 俄语卖点如"Нержавеющая сталь"（不锈钢）、"большая мощность"（大功率）等都会保留
- 只删除俄语品牌名，保留所有产品特性描述

### Q: 左侧的红色信息栏会保留吗？
**A**: 会！对于有信息面板的图片：
- 完整保留信息面板（红色、蓝色等彩色侧边栏）
- 保留面板内的所有文字（任何语言）
- 保持面板颜色和布局
- 只修改产品后面的背景

### Q: 图片尺寸会改变吗？
**A**: 不会！输出图片尺寸与输入完全一致，不裁剪不缩放。

### Q: 文字看不清怎么办？
**A**: 系统会自动调整文字颜色：
- 浅色背景 → 深色文字
- 深色背景 → 浅色文字
- 自动添加阴影或轮廓增强可读性

### Q: 处理时间多久？
**A**:
- 单张：10-20秒
- 10张：2-4分钟
- 50张：10-15分钟
- 复杂布局可能需要稍长时间

### Q: 支持哪些平台和语言？
**A**:
- ✅ 平台：淘宝、京东、拼多多、天猫、OZON等
- ✅ 语言：中文、俄语、英语、日语等所有语言
- ✅ 布局：简单图片、左右分栏、信息面板等
- ⚠️ OZON等国际平台需使用图片直链
